|GDD| vs 带宽 Δλ（当前 N）

展示在当前周期数下，群延时色散随啁啾带宽的变化关系（拟合模型）。红色标记为当前工作点。

计算原理
① 拟合模型（经验验证 k = 5.600×10⁻³ ps²·nm，R² = 0.999999）
|GDD| = k · N / Δλ_nm

② 物理模型（从 Bragg 条件推导）
Λ = λ₀ / (2 · n_eff) — Bragg 周期
L = N · Λ — 光栅总长度
|GDD| = n_g · L · λ₀² / (π · c² · Δλ)

③ 色散符号
· 短波在前（λₘᵢₙ→λₘₐₓ）→ 短波先反射路程短、延迟小 → dτ/dλ > 0 → 反常色散 GDD < 0
· 长波在前（λₘₐₓ→λₘᵢₙ）→ 长波先反射路程长、延迟大 → dτ/dλ < 0 → 正常色散 GDD > 0
